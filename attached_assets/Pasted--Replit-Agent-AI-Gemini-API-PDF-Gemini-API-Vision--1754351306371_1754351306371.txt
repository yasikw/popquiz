クイズアプリ作成プロンプト（Replit Agent用）
以下の仕様でクイズアプリを作成してください：
AI機能（Gemini API活用）

PDFテキスト抽出：Gemini APIのVision機能でPDFを画像として読み込み、テキスト抽出
クイズ生成：抽出したテキストや動画字幕からGemini APIで4択クイズを自動生成
難易度調整：プロンプトエンジニアリングで難易度に応じた問題を生成
解説生成：各問題に対する詳細な解説をGemini APIで自動生成

Gemini API設定

API KEY：環境変数で管理（GEMINI_API_KEY）
モデル：gemini-1.5-pro（テキスト処理・クイズ生成用）
Vision機能：gemini-1.5-pro-vision（PDF画像解析用）
プロンプトテンプレート：
以下のコンテンツから{difficulty}レベルの4択クイズを10問作成してください。
各問題には正解とわかりやすい解説も含めてください。
形式：JSON形式で返答


コンテンツ処理フロー

PDFアップロード → Gemini Vision APIでテキスト抽出
テキストファイル → 直接テキスト読み込み
YouTube URL → 字幕APIで字幕取得
コンテンツ解析 → Gemini APIでクイズ生成（難易度別）
問題保存 → データベースに格納

難易度設定

初級・中級・上級の3段階の難易度選択
選択した難易度に応じて問題の複雑さや専門性を調整
初級：基本的な事実や概念
中級：理解と応用が必要な問題
上級：分析的思考や専門知識が必要な問題

ユーザー管理機能

ユーザー名の登録・変更機能
ユーザー別のデータ管理

スコア・記録機能

各クイズの正答率を記録
回答時間の測定と記録
過去の成績履歴を保存
成績をグラフで視覚化（折れ線グラフ、棒グラフ等）
難易度別の成績統計

UI/UX要件

直感的で分かりやすいインターフェース
スマートフォン・タブレット・デスクトップ対応のレスポンシブデザイン
モダンで親しみやすいデザイン
クイズ中の進捗表示
結果画面での詳細フィードバック

技術要件

フロントエンド：React.js + TypeScript
バックエンド：Node.js + Express
データベース：SQLite（Replit環境に適合）
AI処理：Google Gemini API（PDF読み込み、テキスト解析、クイズ生成）
動画処理：YouTube Data API v3 + YouTube Transcript API（字幕取得）
グラフ表示：Chart.js または Recharts
レスポンシブ：Tailwind CSS
ファイル処理：multer（ファイルアップロード）

画面構成

ホーム画面：ユーザー登録/ログイン、コンテンツアップロード
設定画面：難易度選択、ユーザー情報編集
クイズ画面：問題表示、回答選択、タイマー
結果画面：スコア表示、詳細解答
統計画面：成績グラフ、履歴一覧

データ構造

ユーザー情報（名前、作成日時）
クイズセッション（日時、難易度、スコア、所要時間）
問題データ（質問、選択肢、正解、解説、難易度）

セキュリティ・パフォーマンス

ファイルサイズ制限（PDF: 10MB、テキスト: 1MB）
Gemini API キーの環境変数管理
API レート制限の考慮（Gemini API: 15 RPM）
適切なエラーハンドリング（API呼び出し失敗時の処理）
データの永続化
リクエスト制限とキャッシュ機能

実装のポイント

Gemini API統合：
javascriptconst { GoogleGenerativeAI } = require("@google/generative-ai");
const genAI = new GoogleGenerativeAI(process.env.GEMINI_API_KEY);

PDF処理：multerでアップロード → Gemini Visionで画像解析
プロンプト設計：難易度別のクイズ生成プロンプトを用意
エラー処理：API制限やネットワークエラーに対する適切な処理

開発手順

環境設定：Gemini API KEY設定、必要なパッケージインストール
基本UI：ファイルアップロード画面、ユーザー登録画面
Gemini API統合：テキスト抽出とクイズ生成機能
データベース設計：ユーザー、クイズ、スコア管理
クイズ機能：問題表示、回答記録、タイマー
統計・グラフ機能：成績可視化
レスポンシブ対応：モバイル最適化
